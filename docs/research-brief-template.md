## Official Source Priority

When the task depends on current agent/tool behavior, prefer official or
primary sources first:

- Anthropic Claude Code docs for Claude Code, subagents, hooks, skills, and
  `CLAUDE.md` behavior
- OpenAI Codex docs for `AGENTS.md`, Codex skills, plugins, MCP, and Codex
  runtime behavior
- project-local docs, config, hooks, scripts, logs, and tests for current repo
  truth

If non-official sources are used, explain why they are only supporting context
and not the source of truth.
